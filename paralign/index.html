paralign

Paralign is my third generative NFT project, and my second project on fx(hash). Building upon what I learned while working on gridography, I added more flexibility and functionality. I was also able to explore the new features of fx(hash), which allowed royalty splitting, automated donations, and reserves.

The cornerstone of this project is the imperfect line style. This effect is created using a noise function, that smoothly changes the thickness and curve of the lines.
