overlays

  • view project on objkt

Overlays is my genesis NFT project, a pre-generated collection available on objkt. Written entirely in Python using the Python Imaging Library (PIL), it is quite versatile in the images that it can output. It uses a simple path-finding algorithm to draw two layers of lines, randomly omitting some. Other features affected by randomness are the line thickness, color palette, and line density.
