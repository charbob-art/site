gridography

  • view project on fx(hash)

Gridography is the second art collection I created, and the first project I developed for generation-at-mint on fx(hash). The artwork is a grid of circles, which vary in lightness. Randomness affects the number of rows, columns, and glowing red dots. It is very common for a piece to have no red dots, and extremely rare to have 8 or more dots.

The timing of the release of Gridography was unfortunate, because it was quickly followed by the release of fx(hash) 1.0. While the 1.0 release brought many exciting features, it meant that any collection with an open contract (not fully minted) would be cut off wherever it was in the minting process. This means that gridography wasn't able to contain the intened 88 pieces, and is forever locked at 37.
